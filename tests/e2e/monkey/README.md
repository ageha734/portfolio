# モンキーテスト

このディレクトリには、ランダムな操作や異常系の動作を検証するモンキーテストが含まれています。

## ファイル構成

テストは機能ごとに分割されています：

- **`interactions.spec.ts`**: ランダムなインタラクション（リンク、ボタン、入力フィールド）のテスト
- **`navigation.spec.ts`**: ラピッドナビゲーションのテスト
- **`mouse.spec.ts`**: ランダムなマウス操作のテスト

## テスト内容

### ランダムなインタラクション (`interactions.spec.ts`)

- リンク、ボタン、入力フィールドへのランダムな操作
- スクロール操作
- ブラウザの戻る/進む操作
- アプリケーションがクラッシュしないことを確認

### ラピッドナビゲーション (`navigation.spec.ts`)

- 複数のルート間を高速に移動
- ナビゲーションエラーが発生しないことを確認

### ランダムなマウス操作 (`mouse.spec.ts`)

- ランダムな位置へのクリック
- マウス操作によるエラーが発生しないことを確認

## 実行方法

```bash
# すべてのモンキーテストを実行
bun run e2e -- tests/e2e/monkey

# 特定のテストファイルを実行
bun run e2e -- tests/e2e/monkey/interactions.spec.ts
bun run e2e -- tests/e2e/monkey/navigation.spec.ts
bun run e2e -- tests/e2e/monkey/mouse.spec.ts
```

## 注意事項

モンキーテストはランダムな操作を行うため、実行結果が毎回異なる可能性があります。エラーが発生してもテストは続行され、エラーはログに記録されます。
